# Automate-Bug-Hunting-with-AI-using-MCP-Kali-Server
AI-powered automated web application vulnerability assessment using MCP Kali Server and PortSwigger Labs.
📌 Project Overview

This project demonstrates how Artificial Intelligence can automate web application security testing by integrating MCP Kali Server with Kali Linux tools.AI Automated Bug Hunting using MCP Kali Server is a cybersecurity project that demonstrates how Artificial Intelligence can assist in automating web application security testing. The project integrates an AI assistant with MCP Kali Server, enabling the execution of Kali Linux security tools for reconnaissance, vulnerability analysis, and report generation.

The assessment was conducted on PortSwigger Web Security Academy labs, where the AI successfully identified and validated a Path Traversal vulnerability. The workflow included environment setup, MCP server configuration, AI-assisted scanning, vulnerability detection, manual validation, and documentation of the findings.

This project highlights how AI can enhance penetration testing by reducing manual effort, improving testing efficiency, and assisting security professionals during vulnerability assessments while emphasizing the importance of human verification for accurate results.
